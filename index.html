AMCPRAT: AMC Practice Tool

Get five practice problems of the correct difficulty from the AMC 8, AMC 10, AMC 12, or AIME.

Why use AMCPRAT?

  • Automate practicing problems: Don't worry about choosing problems or checking answers.

  • Easy to use: It's easy to select problems and submit your answers for them.

  • Motivate yourself: It's so easy to practice, so it might be a habit! And enjoy getting problems correct.

  • Open Source: Improve the code base, or leave any issues! View the code here.
